## ✨ Features

- **Animated Hero** — Role switcher with smooth transitions
- **Custom Cursor** — Interactive cursor with hover states (desktop only)
- **Scroll Reveal** — Sections animate into view using IntersectionObserver
- **Animated Background** — Floating blob shapes with CSS animations
- **Professional Journey** — Education timeline (B.Tech, Intermediate, 10th)
- **Badges of Expertise** — Clickable certification cards linking to credentials
- **Interactive Terminal** — Simulated terminal widget in the About section
- **Tech Marquee** — Auto-scrolling core technology strip
- **Featured Projects** — Full-width project showcase with live + GitHub links
- **Technical Skills** — Categorised skill grid (Languages, Frontend, Backend, Databases, Cloud, Tools)
- **GitHub Stats Card** — Language breakdown and commit activity
- **LeetCode Stats** — Difficulty-based problem tracking card
- **Contact Form** — Powered by [FormSubmit](https://formsubmit.co) with success/error feedback
- **Mobile Hamburger Menu** — Slide-down drawer navigation for small screens
- **Fully Responsive** — Optimised for mobile, tablet, and desktop

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) ≥ 18
- [pnpm](https://pnpm.io) (recommended)

### Install & Run

```bash
# Install dependencies
pnpm install

# Start development server (http://localhost:5173)
pnpm run dev
```

### Build for Production

```bash
pnpm run build
```

### Preview Production Build

```bash
pnpm run preview
```

### Lint

```bash
pnpm run lint
```
